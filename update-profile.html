Testing Guide

            
Follow these steps to test file uploads and document analysis:

            
            
    
                
  1. First, update the demo account profile using the "Update Profile" tab.
    2. 
                
  2. Verify all fields are properly set using the "Check Status" tab. All fields should show as "Complete".
    3. 
                
  3. Log in to the application with the sysdemoaccount username.
    4. 
                
  4. Navigate to the document upload section and upload a test file (under 500MB).
    5. 
                
  5. Try a document analysis operation that previously failed with Error 400.

Troubleshooting Tips:

            
    
                
  • If you still get Error 400, check if your upload is completing successfully and the file appears in your account.
    • 
                
  • For larger uploads, monitor the network tab in your browser's developer tools for any timeout issues.
    • 
                
  • If uploads fail with Error 413, refresh the page and try again with a smaller file or try different file formats.
